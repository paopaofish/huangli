# 黄历农历 (Lunar Almanac & Widget)

基于 Kotlin 和 Google 最新推出的 **Jetpack Compose** 与 **Glance (androidx.glance)** 框架，精心打造的一款优雅、精致、现代的「黄历农历」应用和桌面小部件。

该应用不仅能帮助您便捷地掌握农历年月日和传统黄历宜忌，更提供极富东方审美的个性化主题，让中国传统文化与现代 Android 系统美学完美结合。

---

## ✨ 核心特色功能

1. **精准农历与干支推算**
   - 实时显示农历年月日、生肖属相（如：鼠、牛、虎、兔等）。
   - 包含高精度天干地支推算（干支年、干支月、干支日），让传统纪年一目了然。

2. **十二时辰与实时同步**
   - 实时推算当前时间所属的传统**十二时辰**（子、丑、寅、卯等）以及对应的具体时间跨度（如：`23:00-01:00`）。
   - 配合主界面 Live Clock，每 10 秒自动刷新，时刻保证时辰的精准无误。

3. **每日黄历宜忌与动态冲煞**
   - 传承经典黄历算法，依据每日日柱的地支属性，自动推算出当天所适宜（**宜**）与忌讳（**忌**）的日常事项（如：祭祀、祈福、出行、动土、嫁娶等）。
   - 新增**每日冲煞信息**展示（如：“冲兔煞东”），动态关联当日天干地支，并以一致的视觉标签样式融入黄历主卡片及桌面小部件。

4. **小六壬掌中妙算 (Xiao Liu Ren)**
   - 融合传统指心神算，根据当天的农历月份、日期和当前时辰，循环轮推，一键掐指推算当下行止吉凶（大安、留连、速喜、赤口、小吉、空亡）。
   - 提供极富东方韵味、配有流畅展开/收起动画的占卜面板，并加入沉浸式掐指计算等待动效。
   - 提供经典歌诀、精准的推推公式路线图，以及涵盖**有利方位、谋事求财、行人动向、寻人寻物、身体健康、官司纠纷**共六个维度的精细化吉凶解读与现代行止建议。

5. **Glance 桌面小部件 (App Widget)**
   - 使用新一代 **Jetpack Glance** 框架构建，支持在系统桌面一键添加精美小部件。
   - 小部件布局完全由 Compose 代码驱动，触控响应极速，支持长按调整大小并自适应布局。
   - 支持实时显示农历、干支、时辰、六项主宜忌以及最新的**冲煞信息**。
   - 随系统时间半小时自动更新，或通过主应用一键手动刷新，确保桌面信息实时同步。

6. **个性化背景主题 (Personalization Themes)**
   - 内置 5 套具有浓郁东方韵味与极简美感的高级渐变配色主题：
     - **极简流金 (Classic Gold)**：典雅沉稳的深玄底色，辅以灿金点缀。
     - **极简石板 (Modern Slate)**：充满现代科技感的石板灰色，清冷内敛。
     - **赤霄琼脂 (Crimson Jade)**：大气的华夏红，尊贵温润。
     - **水墨禅意 (Zen Ink)**：素雅的宣纸白与徽墨灰，纸香墨飞。
     - **深林苍苔 (Forest Moss)**：静谧的苍翠竹青色，生机勃勃。
   - 主应用内选定后，自动通过本地 SharedPreferences/DataStore 同步，**桌面小部件样式与主应用实时互通**！

7. **1:1 桌面小部件实时预览 (Widget Live Preview)**
   - 在主应用内部直接提供 100% 还原的“桌面小部件实时预览卡片”，更换主题或刷新时可立即直观预览桌面实际效果。

## 🛠️ 技术架构与亮点

- **开发语言**：100% 采用 Kotlin 编写。
- **UI 框架**：采用 **Jetpack Compose (Material 3)** 构建主应用界面，拥有流畅的组件过渡和优雅的卡片流体动画。
- **小部件框架**：引入 Google 最新推荐的 **Jetpack Glance (`androidx.glance`)**。Glance 帮助开发者摆脱复杂的 RemoteViews XML，直接使用类似 Compose 的声明式 API 完美定义小部件视图与点击跳转行为。
- **纯物理算法，100% 离线安全**：
  - 精心优化的万年历高精度算法（支持 1900-2059 百年跨度），**无需连接互联网，不收集任何用户隐私数据**，极度省电、省流、安全。
  - 核心算法文件：`LunarHelper.kt`，经过高精度位掩码位运算测试，确保农历大小月、闰月精准无误。
- **高测试覆盖与稳定性保障**：
  - 配备完整的 JVM 本地单元测试。
  - 使用 **Robolectric + Roborazzi** 组合，提供强大的 Critical User Journeys (CUJ) 场景覆盖以及界面快照/截图比对，确保任何情况下 App 的完美视觉表现与正确性。

## 📱 桌面小部件添加指南

在您的 Android 手机上，通过以下简单步骤即可让本小部件常驻您的桌面：

1. **返回到手机系统主屏幕** (Launcher)；
2. **长按**桌面空白区域或双指捏合，进入桌面整理模式，选择「**小部件** / **卡片** / **Widgets**」；
3. 在小部件列表中找到「**黄历农历**」；
4. 点击并拖拽「**黄历农历小部件**」到您心仪的屏幕位置摆放；
5. 小部件支持**长按边缘缩放**，您可以自由横向或纵向拉伸，它将优雅自适应更丰富的显示区域。
